swipeSnapKinetic

Scroll left or right to move cards and set current

{{ cardflowSnapKinetic.current+1 }} / {{ cardflowSnapKinetic.cards.length }}
{{ card.title }}

none

Only buttons will change current card

{{ cardflowNone.current+1 }} / {{ cardflowNone.cards.length }}
{{ card.title }}

none (as page)

Only buttons will change current page

{{ cardflowNonePage.current/cardflowNonePage.pageSize+1 }} / {{ cardflowNonePage.cards.length/cardflowNonePage.pageSize }}
{{ card.title }}

swipe

Scroll left or right to move cards. No current is tracked. This is sort of like how Hulu mobile app works.

{{ card.title }}

swipeSnapOne

Swipe right or left to scroll active card by 1.

{{ cardflowSnapOne.current+1 }} / {{ cardflowSnapOne.cards.length }}
{{ card.title }}

swipeSnapPage

Like swipeSnapOne, but by the page. This is sort of like how Netflix mobile app works.

{{ cardflowSnapPage.current/cardflowSnapPage.pageSize+1 }} / {{ cardflowSnapPage.cards.length/cardflowSnapPage.pageSize }}
{{ card.title }}

swipeSnap

Scroll left or right to move cards and set current

{{ cardflowSnap.current+1 }} / {{ cardflowSnap.cards.length }}
{{ card.title }}
